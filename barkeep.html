Barkeep!

General Assembly Project:
Web Development for a Cocktail App

Summary

Barkeep! is an interactive cocktail web app that serves up drink recommendations, recipe information and a personal archive of drinks. This was a project done during my stint in General Assembly's Web Development Immersive Program. I built both the back-end webserver/database and front-end user interface.

Technologies Used

Ruby on Rails, PostgreSQL, ActiveRecord, YouTube API, AJAX, JSON, Bootstrap, jQuery, Heroku, Git/GitHub, BCrypt, HTML, CSS, JavaScript
